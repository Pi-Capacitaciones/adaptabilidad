Modelo de caja

El navegador es el interprete de nuestro código HTML, necesita de un modelo para poder mostrar correctamente los elementos que componen nuestra maqueta en la pantalla de nuestro usuario.

El modelo de caja es la interpretación del tamaño de los elementos que componen nuestra maqueta. Las propiedades que vamos a usar controlan el alto, ancho, borde, márgenes externos e internos de nuestros elementos.
